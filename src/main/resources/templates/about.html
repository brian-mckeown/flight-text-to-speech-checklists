About FlightDash.io

FlightDash.io is an open-source, free Electronic Flight Bag (EFB) web application dedicated to Flight Simmers. Our goal is to enhance your flight simulation experience through a range of features:

  • Checklists: Written and readouts for ease of use.
  • Customize and Save: Personalize your checklists and save them for repeated use.
  • Voice Command: Voice recognition checklist responses.
  • Comprehensive Airport Data: Get insights on airport details, weather reports (with configurable METAR sources including VATSIM), frequencies, and runway specifics. Also, get suggestions on in-use runways based on prevailing wind conditions.
  • Flight Plan Synchronization: Seamless integration with SimBrief.
  • Upcoming Feature: Stay tuned for Navigraph Charts in our future releases!
  • Real-Time Flight Data: Monitor your flight's on-time vs. delayed statuses and track the timeliness of each flight phase.
Please Note: While the application is open-source and free, it's strictly designed for flight simulation. Do not use this for real-world aviation!

Developer's Corner

For those interested in the technical bits:

  • Frontend: Angular JS + Bootstrap 5
  • Backend: Java Spring Boot, Thymeleaf, Maven
  • Hosting: Railway.app
  • Integration Services:
    • METAR: aviationweather.gov & metar.vatsim.net
    • Speech-to-text & Voice Recognition: Mozilla Web Speech API
    • Airport Data: airportdb.io API
    • Flight Plan: Simbrief API
  • Code Management: GitHub
